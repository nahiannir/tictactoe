✨ Features

- 🎯 2-player mode (X vs O)
- 🧠 Smart win detection (rows, columns, diagonals)
- ⚖️ Draw detection
- 🚫 Input validation (no invalid/repeated moves)
- 💻 Clean terminal interface

---



🛠️ Tech Stack

- Category: Tech

- Language: Python 3

- Interface: Terminal

- Concepts Used: Lists, Loops, Functions, Conditionals, Exception Handling

---

🧠 How It Works

- The board is stored as a list of 9 elements
- Each index represents a position: (1-9)


- Players take turns:
  - Input is validated
  - Board updates
  - Win/draw is checked
- Game ends when:
  - A player wins 🎉
  - Or it's a draw 🤝

🚀 Getting Started

🔧 Prerequisites

- Python 3 installed

Check:

`python --version`

---

▶️ Run Locally

### Clone the repository
`git clone https://github.com/nahianir/tictactoe.git`

### Navigate into the project
`cd tic-tac-toe`

### Run the game
`python tic_tac_toe.py`

---

🎮 Gameplay Instructions

- Player X starts first
- Enter a number from 1–9
- Players alternate turns
- First to align 3 marks wins

---

⚠️ Rules

- Only numbers 1–9
- No overwriting moves
- Invalid input → retry
